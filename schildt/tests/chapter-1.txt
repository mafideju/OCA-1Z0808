1 - O que é bytecode e porque ele é tão importante para para o uso em Java na programação?
-- Bytecode é o produto gerado após a compilação do código fonte e posteriormente gerado o código binário que é lido pelo sistema operacional do computador.
-- É executado pela JVM e fornece mais segurança e portabilidade para a aplicação.

3 - Onde os programas Java começam a ser executados?
